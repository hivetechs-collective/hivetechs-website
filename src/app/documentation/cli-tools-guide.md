## ğŸ”§ Configuration & Setup

### Provider Setup

### Pipeline Configuration
```bash

  --generator "OpenAI:gpt-4:0.7" \
  --refiner "Anthropic:claude-3-opus:0.5" \
  --validator "Google:gemini-pro:0.3" \
  --curator "OpenAI:gpt-4:0.2"
```
